HTML

¿Qué es HTML?

HTML o su nombre HyperText Markup Language (Lenguaje de Marcado de Hipertexto). Es el lenguaje estándar que se usa para crear páginas web. No es un lenguaje de programación, sino un lenguaje de marcado, porque su función principal es marcar el contenido para que los navegadores web lo interpreten

  • HTML describe la estructura de una página web.
  • Está compuesto por una serie de elementos o etiquetas.
  • Estos elementos le indican al navegador cómo mostrar el contenido.
  • Cada elemento de HTML puede etiquetar partes del contenido, por ejemplo: <p> <div> <a>.
